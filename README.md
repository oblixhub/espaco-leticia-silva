# Espaço Leticia Silva

Landing page institucional para o Espaço Leticia Silva, cabeleireira profissional em Ipiaú, Bahia.

## Recursos

- apresentação dos serviços e portfólio;
- depoimentos e informações de contato;
- formulário que prepara a solicitação e abre o WhatsApp;
- layout responsivo e navegação acessível;
- imagens em WebP com carregamento sob demanda;
- metadados de SEO e cabeçalhos de segurança para a Vercel.

## Desenvolvimento local

Este é um site estático, sem etapa de compilação. Abra `index.html` em um servidor HTTP local ou use a Vercel CLI:

```sh
vercel dev
```

## Publicação

O projeto está configurado para publicação automática pela Vercel a partir da branch principal do GitHub.
